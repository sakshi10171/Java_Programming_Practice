# Java Basic Programs

This folder contains **basic Java programs** written for practicing fundamental programming concepts.

---

## 📂 Programs Included

| File Name | Description |
|----------|-------------|
| HelloWorld.java | Prints "Hello, World!" |
| SumTwoNumbers.java | Calculates sum of two numbers |
| EvenOdd.java | Checks whether a number is even or odd |
| PrimeNumber.java | Checks whether a number is prime |
| PalindromeNumber.java | Checks whether a number is palindrome |
| Factorial.java | Calculates factorial of a number |
| FibonacciSeries.java | Prints Fibonacci series |
| ReverseNumber.java | Reverses a number |
| SumOfDigits.java | Calculates sum of digits |
| LargestOfThree.java | Finds largest of three numbers |
| LeapYear.java | Checks whether a year is leap year |
| CountDigit.java | Counts number of digits |
| Armstrong.java | Checks whether a number is Armstrong |

---

## 📌 Concepts Covered
- Input and Output
- Conditional statements (if-else)
- Loops (for, while)
- Basic mathematical logic
