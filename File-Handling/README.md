# Java File Handling

This folder contains **Java programs demonstrating file handling operations**.  
These programs cover basic to intermediate file operations such as reading, writing, appending, searching, and deleting files.

---

## 📂 Programs Included


| File Name | Description |
|----------|-------------|
| WriteFile.java | Writes data to a file |
| ReadFile.java | Reads data from a file |
| AppendFile.java | Appends data to an existing file |
| WordCountFile.java | Counts number of words in a file |
| CopyFile.java | Copies content from one file to another |
| LineCountFile.java | Counts number of lines in a file |
| SearchWordFile.java | Searches a word in a file |
| DeleteFile.java | Deletes a file |
| CreateFile.java | Creates a new file |
| FileInfo.java | Displays file information |
| RenameFile.java | Renames a file |
| CheckFileExists.java | Checks if file exists |
| ReadFileBuffered.java | Reads file using BufferedReader |

---

## 📌 Concepts Covered
- File creation and writing  
- File reading using Scanner  
- Appending data to file  
- File copying  
- Counting words and lines  
- Searching data in file  
- File deletion  
- Exception handling (try-catch)
