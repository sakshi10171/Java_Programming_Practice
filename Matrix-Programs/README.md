# Matrix Programs in Java

This folder contains basic and commonly asked **Matrix Programs** implemented in Java. These programs help in understanding two-dimensional arrays, matrix operations, nested loops, and problem-solving techniques frequently asked in coding interviews.

## Programs Included

| No. | Program Name | Description |
|-----|---------------------------|-----------------------------------------------|
| 1 | MatrixAddition.java | Adds two matrices of the same order. |
| 2 | MatrixSubtraction.java | Subtracts one matrix from another. |
| 3 | MatrixMultiplication.java | Multiplies two matrices. |
| 4 | TransposeMatrix.java | Finds the transpose of a matrix. |
| 5 | DiagonalSum.java | Calculates the primary and secondary diagonal sums of a square matrix. |

## Concepts Covered

- Two-Dimensional Arrays
- Matrix Operations
- Nested Loops
- User Input using Scanner
- Matrix Traversal
- Matrix Multiplication Logic
- Matrix Transpose
- Diagonal Elements
