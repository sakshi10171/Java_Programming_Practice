# Matrix Programs in Java

This folder contains basic and commonly asked **Matrix Programs** implemented in Java. These programs demonstrate the use of two-dimensional arrays, nested loops, and matrix operations, making them useful for beginners and Java interview preparation.

## Programs Included

| No. | Program Name | Description |
|----:|-----------------------------|----------------------------------------------------------|
| 1 | MatrixAddition.java | Adds two matrices of the same order. |
| 2 | MatrixSubtraction.java | Subtracts one matrix from another. |
| 3 | MatrixMultiplication.java | Multiplies two matrices. |
| 4 | TransposeMatrix.java | Finds the transpose of a matrix. |
| 5 | DiagonalSum.java | Calculates the primary and secondary diagonal sums of a square matrix. |
| 6 | IdentityMatrix.java | Checks whether a matrix is an Identity Matrix. |
| 7 | SymmetricMatrix.java | Checks whether a square matrix is symmetric. |
| 8 | UpperTriangle.java | Displays the upper triangular elements of a matrix. |
| 9 | LowerTriangle.java | Displays the lower triangular elements of a matrix. |
| 10 | RowSum.java | Calculates the sum of each row in a matrix. |
| 11 | ColumnSum.java | Calculates the sum of each column in a matrix. |
| 12 | LargestElementMatrix.java | Finds the largest element in a matrix. |
| 13 | SmallestElementMatrix.java | Finds the smallest element in a matrix. |
| 14 | BoundaryElements.java | Displays only the boundary elements of a matrix. |
| 15 | MatrixTrace.java | Calculates the trace (sum of the primary diagonal) of a square matrix. |
| 16 | SearchElementInMatrix.java | Display the searched element from matrix |
