# Java Array Programs

This folder contains **Java programs based on arrays**.  
These programs help in understanding array operations and logic building in Java.

---

## Programs Included

| File Name | Description |
|----------|-------------|
| ArraySum.java | Finds the sum of elements in an array |
| LargestElementArray.java | Finds the largest element in an array |
| ReverseArray.java | Reverses the elements of an array |

---

## Concepts Covered
- Array declaration
- Taking input using Scanner
- Iterating through arrays using loops
- Basic array operations
